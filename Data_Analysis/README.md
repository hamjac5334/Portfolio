# Data Analysis Projects

Hi, I'm Jack Hamilton. This folder holds the analytics projects I'm most proud of, built alongside my Computer Science degree (Data Analytics minor) at Hendrix College and now while working as a Data/AI Analyst and starting my M.S. in Business Analytics at Georgetown.

**Showcase Projects** [Showcase Projects](https://github.com/hamjac5334/Portfolio/tree/main/Data_Analysis/showcase_projects)

- **[Capital Bikeshare Membership Conversion:](./Capital_Bikeshare)** Gradient-boosted model identifying 55,226 casual DC bikeshare riders who behave like members, sized into a targeted $127K conversion campaign. [Live dashboard](https://public.tableau.com/app/profile/jack.hamilton4441/viz/Capital_Bikeshare_Conversion_Target_dashboard/Dashboard1)
- **[Barcelona 2028: Airbnb Ban Impact:](./BarcelonaAirbnbBan)** Difference-in-differences model using NYC and Lisbon as natural experiments to predict how Barcelona's proposed short term rental ban will affect housing prices. [Live dashboard](https://public.tableau.com/app/profile/jack.hamilton4441/viz/BarcelonaAirbnb_17806215055810/Dashboard1)
- **[Telecom Customer Churn Prediction & Retention Strategy:](./Maven_Telecom_Churn)** Cost sensitive XGBoost model (87% accuracy, F1 .78) tuned to prioritize catching churners over minimizing false positives, informing a targeted retention strategy.
- **[Credit Card Fraud Detection:](./credit_card_fraud)** — SQL and Python fraud detection pipeline built on Capital One's own public fraud dataset, tackling extreme class imbalance with SMOTE, resampling, and ensemble classifiers.
- **[Semantic Evaluation of UN SDG Reports:](./SDG_Semantics)**  NLP analysis of how UN Sustainable Development Goal report language shifted from 2016–2025, validating an urgency metric against real UN progress ratings.
