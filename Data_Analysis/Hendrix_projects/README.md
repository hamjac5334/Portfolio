
# Scientific Computing Work: [my portfolio site](https://hamjac5334.github.io/projects.html)

Labs and projects from my Scientific Computing coursework at Hendrix College, applying simulation, optimization, and statistical modeling to real-world questions. Fuller write-ups for select projects live on.

- **[FloridaGerrymanderingWithTSP.ipynb](./FloridaGerrymanderingWithTSP.ipynb)** — Optimized Florida's congressional district maps to reduce gerrymandering, using TSP-style optimization and quantitative fairness metrics.
- **[jack-hamilton-heart-lab.ipynb](./jack-hamilton-heart-lab.ipynb)** — Built predictive models for heart failure risk from patient health indicators, with exploratory analysis and model evaluation.
- **[lab-5-predicting-the-mid-term-elections.ipynb](./lab-5-predicting-the-mid-term-elections.ipynb)** — Predicted 2024 U.S. Senate election outcomes using Monte Carlo simulation over polling data.
- **[malaria-project-jack-hamilton.ipynb](./malaria-project-jack-hamilton.ipynb)** — Modeled malaria transmission dynamics in Rwanda with an SIR compartmental model, simulating 5-year trajectories under different vaccination scenarios.
- **[nyc-death-data-jack-hamilton.ipynb](./nyc-death-data-jack-hamilton.ipynb)** — Analyzed NYC mortality data for trends across time, cause of death, and demographics.
